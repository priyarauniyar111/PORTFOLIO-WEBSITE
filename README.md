# Portfolio Website
A responsive personal portfolio website built using HTML, CSS, and JavaScript.
It showcases sections like Home, About, Projects, and Contact with a clean UI.
## Features
- Responsive navbar and hero section
- About section with structured content
- Cards layout using Flexbox and Grid
- Contact page with responsive form
- Smooth hover and animation effects
## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- Git & GitHub
## Project Structure
project-name/
│── index.html
│── about.html
│── project.html
│── contact.html
│── css/
│   └── style.css
│── js/
│   └── script.js
│── assets/
│   └── images/
└── README.md
From creating this project structure, you have learned:

How to organize a web project properly

The role of different files:

index.html → Home page

about.html, project.html, contact.html → Multiple pages

css/style.css → Styling the website

js/script.js → Adding interactivity

assets/images/ → Storing images

How to add new pages (project.html) to an existing project

Basic understanding of frontend project structure

How real projects are arranged for GitHub / internships / placements
